# Images

Denna mapp ska innehålla PWA-ikoner:

- **icon-192.png** - 192x192 pixels
- **icon-512.png** - 512x512 pixels

## Skapa ikoner

Du kan skapa ikoner med:
- [favicon.io](https://favicon.io/emoji-favicons/)
- [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)
- Eller valfritt grafikprogram

## Tips

- Använd mörk bakgrund (#0a0a0a) med rosa accent (neonrosa/magenta)
- Se till att ikonerna är tydliga i både ljust och mörkt läge
- Exportera som PNG med transparent bakgrund eller matchande färg
